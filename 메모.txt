적용 취소, 적용에 대한 설명 - 주석
/* ~ */

선택자
	- 태그 선택자
		(태그명으로 선택)
	* -> 전체 태그 선택자 
	- 클래스 선택자
		.클래스명(마침표)
	- 아이디 선택자
		#아디명

- 태그의 style 속성으로 정의 하는 방법
		
적용범위 
태그 > 클래스 > 아이디 > 태그의 style속성을 정의

CSS 적용 우선 순위
(적용 범위가 좁을 수록 우선 순위가 높다)
태그의 속성 style > id > class > 태그

우선순위 강제 적용범위
!important -> 우선순위 강제 적용

조상 자손 선택자
왼쪽 -> 오른쪽(왼쪽으로 갈수록 상위 태그, 오른쪽으로 갈수록 하위 태그)

부모 자식 선택자
바로 위 층위, 바로 아래 층위
부모 > 자식

형제 

인접형제(+)
선택적1+선택자2 -> 선택자1(왼쪽 위치)을 인접 형제로 가진 선택자2를 선택

일반형제(~)
선택자1~선택자2 -> 선택자1을 형제(왼쪽 위치)로 가지고 있는 선택자2를 선택

그룹 선택자
콤마(,)로 구분하여 여러 선택자를 사용

가상 클래스 선택자
	선택자: 상태값
- 요소의 상태로 선택자
	예1) (체크, 라디오 박스 -> 체크가 되었는지?)
		선택자: checked
	예2) 요소에 마우스가 올려 있는 상태인지?
		선택자: hover
	예3) 요소가 읽기 전용 상태인가?
		선택자: read-only
	예4) 선택자가 아닌 상태 인가?
		선택자1에 선택자2가 포함이 되어 있지 않은 상태인가?
		선택자1: not('선택자2')
	예5) 첫번째 자식인가?
		선택자: first-child
	예6) 마지막 자식인가?
		선택자: last-child
	예7) 몇번째 자식인가?
		선택자: nth-child
		nth-child(숫자) -> n번째 요소
		nth-child(수식) -> 수식 계산후 선택
		2n -> 2,4,6,8,10
		
		2n+1 ->1,3,5,7,9
		(n은 0부터 시작)
		
- 가상으로 추가하여 선택하는 선택자
	선택자: after
		- 선택자의 자손으로 가장 끝에 추가

	선택자: before
		-선택자의 자손으로 가장 앞에 추가

속성 선택자
선택자[속성명='속성값']{}

스타일 속성

선택자{
	background-color: red; width: 100%;
	color:blue;
}

속성명 : 속성값;
1. 글꼴 속성
	font-size - 글자의 사이즈
					고정 사이즈
				- px (고정 화소 기준 사이즈)
				- pt (고정 사이즈)
				
					상대 사이즈
				- em - 부모 기준 요소에 기준 사이즈
						부모 13px;
						자식 1em ->  13px	
							  1.2em -> 15.6px 
				- rem - 최상의 태그 기준 사이즈	
						html 13px
						1rem -> 13px
						1.2rem -> 15.6px
	color - 글자 색
			1) 색상명 (red, orange, pink...)
			2) RGB(0~255,0~255,0~255)
			3) hex 코드(16진수)
				rgb 숫자 -> 16진수로 변경
				rgb(255,0,0) -> #ff 00 00
					
text-align - 텍스트 라인 (left-왼쪽 정렬,right-오른쪽 정렬,
								center-중앙 정렬,justify-양쪽 정렬)
								
line-height - 행간 (기본 값 1.2)
letter-spacing - 글자 자간 (글자 사이사이)
word-spacing - 단어별 자간
font-weight - 글자의 두께
				- 문자 (normal - 기본값,  bold - 두껍게)
				- 숫자 (100~900) - 지원되는 폰트에 따라 다르다
font-family - 글꼴
					font-family: 기본 글꼴, 대체 글꼴1, 대체 글꼴2
									기본 글꼴이 없는 경우 차례대로 대체 글꼴이 적용
					시스템 폰트
					
					웹폰트(시스템에 없는 폰트)
					<link> 스타일 css 추가 방식
					@import css 추가
					
2. 공간
	display - 
			block - 공간
						- 너비, 높이 O
						- 상하좌우 여백(내부, 외부)
						- padding(내부 여백)
						- margin(외부 여백)
			inline - 글의 속성
						- 너비, 높이 X
						- 상하좌우 내부 여백 가능(padding)
						- 좌우 외부 여백 가능(margin)
			inline-block
					- inline과 block을 절충
						- 줄 개행X
						- 너비, 높이 O, 모든 방향 외부 여백
			
			none - 감춤 처리
			
			표와 같이 만들어주는
			table - 테이블 행
			table-cell - 테이블 열
			
3. 레이아웃
	box-sizing
		1) 내용 영역 기준
			content-box
			내용 영역의 사이즈를 유지 하면서 확대
			총 사이즈 = 내용 영역 + 내부 여백 + 경계
		
		2) 경계 영역 기준
			border-box
			경계를 기준으로 사이즈를 정하는 방식
			경계 + 내부 여백에 따라서 내용 영역의 사이즈가 변경이 된다.

4. 여백
	1) 내부 여백
		padding
			- padding: 10px; -> 상,하,좌,우(모든방향) 10px
			- padding: 10px 20px; -> 10px(상,하) 20px(좌,우)
			- padding: 10px 20px 30px; -> 10px(상) 20px(좌,우) 30px(하)
			- padding: 10px 20px 30px 40px;
						-> 10px(상) 20px(우) 30px(하) 40px(좌)
			
	2) 외부 여백
		margin
			- padding과 적용 순서 동일
5. 경계
	border - 경계
		
		border: 라인 굵기(px) 경계선 종류 색상:
				경계선 종류
					1) solid - 직선
					2) dotted - 점선
					3) dashed - 파선
				
	border-radius
			-> 경계를 둥글게 (px, %)
			-> 50% = 원
			
6. 포지션
	- 배치
	position
		- static - (기본값)
		
		상대 위치 
		- relative
			- 기준 (요소 자신)
			- 자기자신이 위치한 위치기준으로 상대적으로 이동
		- absolute
			- 기준 (상대위치 부모 또는 조상)
					- 부모, 조상 중 누구라도 상대 위치 속성을 가진다면
						(relative, absolute, fixed) 
					- 기준(relative, absolute, fixed)  내에서 상대적으로 이동
					-상대 속을 가지고 있는 부모가 없다면 전체 문서 기준(html)
						
		- fixed
			- 기준 (물리적으로 보이는 화면 기준으로 상대적으로 이동)
		
		상대위치를 상대적으로 이동하기 위한 것
		속성 존재
		+
		top -> 위에서 아래로
		right -> 오론쪽에서 왼쪽으로
		bottom -> 아래에서 위로
		left -> 왼쪽에서 오른쪽으로
		z-index - 각 요소의 층위
				숫자가 클수록 앞쪽에 있고
				숫자가 작아질수록 뒤쪽에 있다.
			예) z-index: 10, z-index: 9; 
				10이 9보다는 앞에 있다.	
7. 배치
float
	left: 왼쪽 -> 오른쪽
	right: 오른쪽 -> 왼쪽
	
	- float 속성이 업더라도 다음 요소에 영향을 미친다.
	- clear 속성을 반드시 적용해야 한다.
	
clear 
	- 공간 속성을 가진 요소에서 적용 가능
	- left
		float: left 속성 제거
	- right
		float: right 속성 제거
	- both
		float: left, right 속성 모두 제거
		
	가상 클래스 선택자 - after
		예시
		.wrap:after{
					clear:left;
					content: '';
					display: block;
				}
	
	width - 너비
			- calc() -> 계산
			예) width: calc(100% - 200px);	
				width: calc(100% / 3);
				연산 기호 앞 뒤로 반드시 띄어쓰기
				
8. 그래픽
background
	- background-color - 배경색
								- 1) 색상명 2)rgb(0,0,0) 3) hex(#000000)
								- 4) rgba
										a - 알파채널(투명도, 0~1)
										 0에 가까울수록 투명, 1에 가까울수록 불투명
	
	- background-image - 배경 이미지
			background-image:url('이미지 경로');
			
	- background-repeat
		- no-repeat -> 반복 없이 1번만 출력
		- repeat-x -> 왼쪽에서 오른쪽으로 반복 출현
		- repeat-y -> 위에서 아래로 반복 출현
		
	-background-position
			- 배경 이미지의 위치
			background-position: 좌중우(left|right|center) 상중하(top|bottom|center)
			
			수치 입력시
			background-position: 100px     100px;
										(좌->우) (위->아래)
			
	- 단축
	background: 색상, 이미지, 반복, 위치
				(색상, 이미지는 둘다 나와도 되고, 둘중에 하나만 있어도 된다.)
			예) background: #d5d5d5 url("img/logo.png") no-repeat center center;
